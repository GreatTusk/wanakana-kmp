fun toHiragana(input: String, imeMode: IMEMode = IMEMode.DISABLED, passRomaji: Boolean = false, useObsoleteKana: Boolean = false): String

Converts input to Hiragana.

Return

the converted text.

For example:

  • toHiragana("toukyou, オオサカ") // => "とうきょう、　おおさか"`

  • toHiragana("only カナ", passRomaji = true) => "only かな"

  • toHiragana("wi") => "うぃ"

  • toHiragana("wi", useObsoleteKana = true) => "ゐ"

Parameters

input

the text to convert.

imeMode

if enabled, handles conversion while the text is being typed, defaults to IMEMode.DISABLED.

passRomaji

if true romaji will be kept as-is, defaults to false.

useObsoleteKana

if true obsolete kanas will be used (ゐ and ゑ), defaults to false.
