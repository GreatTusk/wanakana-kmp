More info  


Sets [useObsoleteKana](use-obsolete-kana.md). If true, obsolete kanas (ゐ, ゑ, ヰ and ヱ) will be used when converting romaji to kana. Applies to toKana, toHiragana and toKatakana. Defaults to false.
